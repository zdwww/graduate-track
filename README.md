# GraduateTrack

## Project Objective

GraduateTrack is a graduate school application tracker. It helps applicants organize programs, deadlines, application statuses, interviews, and academic networking contacts in one place.

## Planned Features

- Create, view, edit, delete, and filter graduate school applications
- Track deadlines, interview dates, application statuses, and notes
- Create, view, edit, and delete professor, alumni, and admissions contacts
- Store user accounts and keep each user's data private

## Planned Work Split

- Team member 1: full-stack Applications user story
- Team member 2: full-stack Contacts user story

## Tech Stack

- Client-side React with Hooks
- Node.js and Express
- MongoDB with the official Node.js driver
- No Mongoose
